## <a href="https://search-user-og2g.vercel.app/" target="_blank">Search User</a>

Приложение состоит из главной страницы, отображающей карточки пользователей. При вводе имя пользователя в инпут и нажатии кнопки Искать показывается карточка выбранного пользователся

Для получения пользователей использовал jsonplaceholder

## Технологии

* React
* Redux-toolkit
* TypeScript
* SCSS
* Webpack 
* Jest/RTL

Клонировать репозиторий:

```
https://github.com/Nikita1025/SearchUser.git

```
Установить пакеты:

```
npm install
```

Запустить проект:

```
npm run start
```

Запустить сборку для продакшена:

```
npm run build:prod
```

Запустить сборку для разработки:

```
npm run build:dev
```
